# ICPC Training 

This repository tracks my training, solutions, and algorithmic templates as I prepare for the **International Collegiate Programming Contest (ICPC)**. 

* **Current Status:** Training for 2026
* **Primary Language:** `C++17` 
* **Secondary:** `Python 3` For math and strings

## Algorithmic Frameworks

My solutions and notes are categorized by core algorithmic concepts. 

- **[Data Structures](./data-structures)**: Trees, DSU, Fenwick, Segment Trees.
- **[Graphs](./graphs)**: BFS/DFS, Shortest Path (Dijkstra, Bellman-Ford), MST, Network Flow.
- **[Dynamic Programming](./dp)**: Knapsack, Digit DP, DP on Trees, Bitmask DP.
- **[Mathematics](./math)**: Number Theory, Combinatorics, Sieve, Modular Arithmetic.
- **[Strings](./strings)**: KMP, Trie, Suffix Automaton, Z-Algorithm.
- **[Geometry](./geometry)**: Convex Hull, Line Intersections, Sweep Line.
